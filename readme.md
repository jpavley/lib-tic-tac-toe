# lib-ttt

## Library Tic Tac Toe

### Overview

Library of tools to implement a robust tic tac toe like game in Apple's 
Swift programming language. Xcode, iOS, or macOS  not required. 

See _Install_ and _Usage_ for notes on how to work with Swift on the 
command line.
